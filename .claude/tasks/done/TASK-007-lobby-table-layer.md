## 1. 개요

### 1.1 문제 정의
로비(Home 씬)에서 테이블이 캐릭터 아래에 렌더링되어 캐릭터가 테이블 위에 있는 것처럼 보입니다.

### 1.2 목표
- 테이블이 캐릭터 위에 렌더링되도록 레이어 순서 수정
- 캐릭터가 테이블 뒤에 있는 것처럼 보이도록 수정

---

## 2. 수정 상세

### 2.1 대상 파일
`assets/Game/Scenes/Home.fire`

### 2.2 관련 노드
- `ui_img_lobby_deco_table` (428번 라인)
- `ui_img_lobby_deco_table2` (616번 라인)

### 2.3 수정 방법

#### 방법 1: zIndex 수정 (권장)
```json
// Home.fire에서 테이블 노드 찾아서 zIndex 값 증가
"_zIndex": 100  // 캐릭터보다 높은 값으로 설정
```

#### 방법 2: siblingIndex 수정
```json
// 테이블 노드를 캐릭터 노드보다 뒤에 배치
"_siblingIndex": 10  // 캐릭터보다 높은 값
```
