# 코드 리뷰

게임 코드 품질을 검사하고 개선점을 제안합니다.

## 검사 항목

- 버그 및 오류 (null 참조, 타입 불일치)
- 성능 이슈 (메모리 누수, 불필요한 연산)
- 보안 취약점 (하드코딩된 키, XSS)
- 코드 품질 (중복, 복잡도, 네이밍)

## 에이전트 실행

```bash
cd agents && npm run code-review
```

## 인자

- `$ARGUMENTS`: 대상 파일/폴더 경로 또는 검사 유형
